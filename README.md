# Hi, I'm Ethan Webster

Front-end developer based in Port Orchard, WA, recently certified through
**Western Governors University** — coming to software from a hands-on technical career.

For years I ran machines and led crews in manufacturing and food-safety operations:
programming CNC laser systems, reading engineering blueprints to tight tolerances, and
supervising a 22-person sanitation team under USDA inspection. That kind of work runs on
precision, reliability, and staying level-headed when something breaks mid-shift — the
same habits I bring to writing and debugging code.

These days I build web apps with JavaScript and React, and I'm working my way toward full-stack.
